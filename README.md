📝 Online Webinar Registration Form

📌 Description

📝 This is a basic **HTML Registration Form** designed for users to register for an online webinar.  

🎯 It collects essential details such as participant name, email address, phone number, organization details, and webinar preferences.  

💻 The form provides a simple and user-friendly interface for webinar registration.

🛠️ Technologies Used

🌐 HTML5 – Form Structure and Validation

✨ Features

👤 Input Full Name  
📧 Email Validation  
📱 Phone Number Input  
🏫 College / Organization Field  
📚 Select Webinar Topic  
📅 Choose Preferred Date  
🎥 Select Mode (Live / Recorded)  
🔄 Reset Form Option  
✅ Required Field Validation
