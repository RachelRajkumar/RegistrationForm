📝 Online Webinar Registration Form

📌 Description
This is a basic HTML Registration Form designed for users to register for an online webinar.
It collects essential details like name, email, phone number, and preferred webinar options.

🛠️ Technologies Used
HTML5

✨ Features
👤 Input Full Name
📧 Email validation
📱 Phone number input
🏫 College / Organization field
📚 Select webinar topic
📅 Choose preferred date
🎥 Select mode (Live / Recorded)
🔄 Reset form option
✅ Required field validation
